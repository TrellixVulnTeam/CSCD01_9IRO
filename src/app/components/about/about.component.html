About the Team

An inside look at Team 6.

Our Strengths

Each team member has strengths which complement the weaknesses of other team members. We have team members who have experience in developing for open source projects, and we all have a lot of experience using python, so we will be able to help each other if one of us gets stuck on something.

Our Goals

Our goals for this project are to make positive contributions to the open source project which accomplishes all of the goals required in an easy to use and efficient manner. We hope to create features which are beneficial to the overall project so that they will be used in the master branch, while also finding bug fixes which are also beneficial.
